    “此管理单元不能用于这一版本的Windows10”，我该如何添加新的Microsoft账户？

    我在点击设置中的”将其他人添加到这台电脑“后弹出“本地用户和组”并说明此管理单元不能用于这一版本的Windows10。版本为联想正版家庭版，除了升级专业版或者重置外，我该如何添加新的Microsoft账户？

    您好，

    了解到您的问题。

    如果您想要调用本地用户和组这个功能组件，那么您只能够通过升级专业版解决该问题。

    如果您只是想再系统下再添加一个本地管理员账户，您可以通过一下方法来执行操作。

    在命令提示符（管理员）中复制粘贴输入以下指令：

    net user User ****** /add（User是新建账户，******是新建的随机密码）

    net localgroup administrators User /add（赋予该账号管理员权限）

    注意：上述方法是添加本地管理员账户的方法，如果要使用微软账户登录系统的，那么您在使用新的本地管理员账户后再将其余微软账户关联即可。

    希望以上信息可以帮助到您。
